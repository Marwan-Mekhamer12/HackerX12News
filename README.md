# HackerX12News

HackerX12News is an iOS news reader app that fetches the latest articles from **Hacker News**. It displays the titles of the top front-page stories in a table view. Tapping a title opens the article in a **WebView** using **WebKit**, allowing users to read the full content directly within the app.

## Features
- Fetches top Hacker News stories using the Hacker News API.
- Displays titles in a table view.
- Opens selected articles in an in-app browser using `WKWebView`.
